# explain-simply

> Claude Code용 스킬 — 방금 한 작업을 **비전문가 눈높이로 다시 풀어 설명**해 주는 도구.

## 왜 만들었나

Claude이나 Codex가 뭔가 만들어주고 나면, **이해 없이 그냥 넘어가는** 일이 자주 생깁니다. 심지어 내가 직접 만든 것조차 나중에 보면 *"이게 뭘 만든 거지, 어떻게 만든 거지, 왜 만든 거지"* 막막해질 때가 있죠.

그럴 때 코드를 한 줄씩 뜯어보는 건 부담스럽습니다. 필요한 건 **"방금 뭘, 왜, 어떤 식으로 했고, 그래서 뭐가 달라지는지"**를 쉬운 말로 한 번 짚어주는 것뿐입니다. 그걸 자동으로 해주려고 이 스킬을 만들었습니다.

간단한 스킬이지만, 만들고 보니 **자주 쓰게 됩니다.**

## 무엇을 해주나

스킬을 발동하면, 직전에 한 작업을 이 순서로 설명합니다:

1. **한 줄 요약** — "방금 한 건 이겁니다"
2. **무엇을 / 왜 / 어떻게 / 그래서** — 무엇을 건드렸고, 왜 필요했고, 어떤 방식으로 했고, 그래서 뭐가 달라지는지
3. (필요할 때만) 핵심 코드 조각을 **짧게** — 줄별 해설은 하지 않습니다
4. 직접 **확인하는 방법**

억지 비유 없이, 전문용어는 그 자리에서 풀어서 설명합니다.

## 어떻게 쓰나

작업 직후 이렇게 말하면 됩니다:

```
쉽게 설명해줘
```

또는 `못 알아듣겠어`, `방금 뭐 한 거야`, `/explain-simply` 등으로도 발동됩니다.

## 누구에게 좋나

코딩·AI 도구를 **처음 접하는 분**에게 특히 좋습니다. 매번 이 스킬로 "방금 뭘 했는지"를 쉬운 설명으로 받아보면서 **자연스럽게 익혀갈 수 있습니다.**

## 설치

`SKILL.md`를 Claude Code의 스킬 디렉토리에 두면 됩니다:

```
~/.claude/skills/explain-simply/SKILL.md
```

Claude Code가 자동으로 인식합니다.
